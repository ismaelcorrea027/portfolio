    Totalmente Gratuito

    Compartilhamento

    Divulgue, compartilhe suas ofertas com seus clientes e amigos.

    Clientes

    Construa sua base de clientes sólida e engajada com suas ofertas. O sistema alimenta automaticamente a base de dados.

    Loja Virtual

    Você poderá criar uma loja virtual para seu site sem precisar ter conhecimentos avançados.

    SEO (seu site no Google)

    Configure facilmente seu site para que ele apareça na busca do Google.

    Cartões de crédito

    Pagamentos via cartões de crédito, incluindo compras realizadas por estrangeiros.

    Passeios / Receptivos

    Compras avulsas onde o SEU CLIENTE define a quantidade de PAXs e a data de ida. Ideal para: Turismo receptivo, passeios e destinos que faz sob demanda.

    Domínio personalizado

    Seu site com o nome personalizado do jeito que você quiser (www.seu-site.com.br)

    Pacotes

    Crie pacotes para GRUPOS e defina o roteiro, configure a quantidade de PAXs, data de ida e volta.

    Boleto bancário

    Pagamentos via boleto, com repasse direto para sua conta bancária.

    Cupons de desconto

    Ofereça descontos para seu cliente, incentivando a conversão da compra.

    Tickets / Ingressos

    Venda on-line de ingressos/tickets para todos os tipos e tamanhos de eventos. Ideal para: Shows, workshops, palestras, cursos, etc.

    Link de pagamento

    Inclua em pedido e envie o link para seu cliente efetuar o pagamento.

    Site Mobile

    Site perfeitamente adaptável para os dispositivos móveis.

    Controle Financeiro

    Controle de receitas e despesas de todas as ofertas.
